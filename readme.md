# YouTube Clone Project

This project is a static clone of the YouTube homepage, created using HTML and CSS.

## Description

The goal of this project was to replicate the look and feel of the YouTube homepage, focusing on layout, positioning, and styling with pure CSS. It includes a responsive grid of video previews, a header with search functionality, and a navigation sidebar.

## Features

*   **Header:** A fixed header containing the YouTube logo, a search bar, and user-related icons.
*   **Sidebar:** A vertical navigation bar with links to different sections like Home, Explore, and Subscriptions.
*   **Video Grid:** A responsive grid layout that displays video thumbnails, titles, channel information, and stats.
*   **Tooltips:** Hover-activated tooltips for icons in the header.

## Technologies Used

*   HTML5
*   CSS3
